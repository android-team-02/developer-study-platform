## 프로젝트

### 소개

- 개발 분야 및 언어를 지정한 스터디를 통해 서로의 지식 공유
- 개발자들 간의 네트워크 형성 도모
- 깃허브 로그인으로 서비스 이용

### 기술 스택
- Kotlin, Retrofit, Coroutine, Firebase, Room, DataStore, Jetpack Navigation, MVVM, DataBinding, ViewModel, LiveData, Flow
